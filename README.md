# [暧昧期的信号解码——TA到底喜不喜欢我？](https://hoo.be/ytqrsp)

暧昧期最是撩人，也最是磨人。那些忽明忽暗的信号，常常让我们心神不宁。

核心观点：行动远比语言更有说服力。

积极信号：

主动发起联系： 主动找你聊天，分享生活琐事。

愿意投入时间： 即使很忙，也会为你留出时间。

身体语言的靠近： 不经意的触碰、眼神的停留、模仿你的动作。

深度自我表露： 与你分享内心想法、过去的经历甚至脆弱面。

消极信号（需警惕）：

聊天回避感情： 永远停留在表面，对你的试探避而不谈。

约会难以推进： 只在深夜找你，或从不计划正式约会。

在社交圈“隐形”： 从不带你进入他的朋友或生活圈子。

专家建议： 如果你收到了足够的积极信号，不妨勇敢地、有策略地向前推进一步。如果长期只有模糊不清的语言而没有实质行动，请做好“对方可能只是享受暧昧”的心理准备，及时止损。
